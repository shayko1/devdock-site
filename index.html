🚀

Project Launchpad

Auto-scans your workspace and detects projects across Node.js, Scala, Go, Rust, Python, Ruby, and more. Start, stop, and monitor dev servers with one click. See which ports are in use system-wide.

💬

Embedded Claude Sessions

Launch Claude Code directly inside DevDock with full terminal emulation. Supports git worktrees for isolated changes, session resume, image paste, clickable links, and a "waiting for input" indicator.

⚙️

Autonomous Pipeline

Describe a task and let the pipeline plan, implement, validate, and review it automatically. Each stage runs in isolated git worktrees. Failed validations trigger automatic retries with feedback.

🤖

Agent Monitoring

Auto-discovers Claude agents from LaunchAgent plists. Shows real-time status, schedules, run history, and logs. Trigger manual runs and view output — all from DevDock.

🌐

Built-in Browser

Claude can control a real browser window for visual testing. Navigate, screenshot, click, type, and inspect pages — all via simple CLI commands injected automatically into each session.

📂

File Explorer & Search

Browse project files, view syntax-highlighted code, search across codebases, and inspect git diffs — all in a side panel next to your Claude session. No context switching needed.
